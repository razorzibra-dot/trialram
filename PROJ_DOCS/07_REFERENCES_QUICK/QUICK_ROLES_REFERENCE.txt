║                      FILES TO MODIFY ROLES                                 ║
╚════════════════════════════════════════════════════════════════════════════╝

1. ADD NEW PERMISSION:
   → src/services/rbacService.ts
     Add to: mockPermissions array

2. CREATE NEW ROLE:
   → src/services/rbacService.ts
     Add to: mockRoles array
   → src/config/navigationPermissions.ts
     Update: roleHierarchy if needed

3. CHANGE ROLE PERMISSIONS:
   → src/services/rbacService.ts
     Edit: permissions array in the role object

4. UPDATE NAVIGATION ACCESS:
   → src/config/navigationPermissions.ts
     Update: requiredRole or permission fields
